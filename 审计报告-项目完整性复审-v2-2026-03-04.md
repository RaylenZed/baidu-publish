## 3. 高优先级问题（阻塞“完整完工”）

### P0（上线阻塞）

1. **`CORS_ORIGINS` 环境变量解析会导致应用启动失败**
   - 现状：`Settings.CORS_ORIGINS` 定义为 `list[str]`，但未做逗号串兼容解析。
   - 复现：当环境变量为 `http://localhost:5173,http://localhost:80` 时，抛 `SettingsError`。
   - 影响：部署环境常见写法会直接阻断后端启动。

2. **运行时引入 `pytz`，但依赖未声明**
   - 现状：`task_service.py`、`schedule_service.py` 使用 `import pytz`；`requirements.txt` 未包含 `pytz`。
   - 影响：涉及每日限额/定时调度计算时可能 `ModuleNotFoundError`。

### P1（核心功能偏差）

1. **账号接口契约与前端实现不一致，多个页面会异常**
   - 后端 `GET /accounts` 返回 `list`；前端按分页结构读取 `res.data.items/total`。
   - 前端还调用了后端未实现的 `GET /accounts/{id}`。
   - 影响页面：账号管理、任务创建弹窗、定时任务页面。

2. **任务创建“品类可留空随机”未实现**
   - PRD：手动创建任务 `category` 可留空，按账号绑定品类随机。
   - 现状：`CreateTaskRequest` 允许空；`TaskService.create_task` 强校验 `category in account.categories`，空值会失败。

3. **手动发布失败状态可能被事务回滚，无法稳定落库 `publish_failed`**
   - `article_service.manual_publish` 在失败路径先改状态再抛异常；
   - 路由层异常触发 `get_db` 回滚，失败状态和中间状态存在被回滚风险。

4. **导入账号的逐条失败处理存在事务副作用**
   - `AccountService.import_accounts` 循环内异常时 `db.rollback()`，会回滚前面未提交成功项，返回统计可能失真。

5. **前端关键 PRD 能力未交付**
   - 未见任务卡片/步骤进度组件、实时日志抽屉、文章编辑器与 diff 组件（`components/` 目录为空）。

## 4. 中优先级问题（影响验收一致性）

1. `App.vue` 菜单缺少“系统日志”，但路由已存在 `system-logs`。
2. `DashboardView.vue` 的账号健康字段按 `id/name` 读取，而后端返回 `account_id/account_name`。
3. `PoolsView.vue` 把后端返回当作按 pool_type 键控对象使用，但后端返回列表。
4. `viewLogs/viewArticle` 直接 `window.open('/tasks/{id}/logs')`、`window.open('/articles/{id}')`，前端无对应路由页。
5. 宪法要求“前端不硬编码品类”，账号页仍保留本地硬编码降级列表。
6. `content_events` 中声明的 `task_created` 事件未在任务创建时实际写入。
7. README 与实际存在偏差（如 WS 路径示例、部署 env 路径、备份命令用户等）。

## 6. 建议修复顺序（最短闭环）

### 第 1 批（必须先做，1 天内）

1. 修复 `CORS_ORIGINS` 解析（支持逗号串与 JSON 数组双格式）。
2. 在后端依赖中补充 `pytz`（或改为 `zoneinfo` 并统一实现）。
3. 统一 `/accounts` 契约：后端改分页响应或前端改按列表读取；同时补齐/移除 `GET /accounts/{id}` 调用。
4. 修复任务创建空品类逻辑（空值时按账号绑定品类随机）。
5. 修复 `manual_publish` 失败时状态落库事务（确保 `publish_failed` 与审计可提交）。

### 第 2 批（验收达标，2-3 天）

1. 完成前端关键 PRD 交互：任务步骤可视化、实时日志、文章编辑/diff。
2. 修复 Dashboard/Pools/SystemLogs 等前后端字段契约偏差。
3. 补齐 `task_created` 事件写入与审计查询能力。
4. 清理并同步 README/部署文档与实际行为。

### 第 3 批（上线前，持续）

1. 增加最小可用测试集：认证、账号、任务创建、发布失败回滚、调度触发。
